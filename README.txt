This is a readme file for a Python project that uses Tkinter and MySQL.
It includes the necessary libraries.
customtkinter
Pillow
mysql-connector
mysql-connector-python
python-dotenv  # optional, if you want to store DB credentials in a .env file

You can install these libraries using pip: pip install customtkinter Pillow mysql-connector mysql-connector-python python-dotenv
To run the project, run the main.py file in the project directory.
The project is currently set to run using a database hosted on Google Cloud Platform.
To run the database locally, run the databasefinalproject.sql file, and chnage the  login details in the connect.py file, in the db folder.
